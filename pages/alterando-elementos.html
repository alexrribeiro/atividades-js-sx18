Estruturas Condicionais e Alteração de Classes

Mudando as cores

Através dos radio buttons a seguir é possível escolher as cores das barras e detalhes desta página. Esta é uma atividade para demonstrar a mudança de classes (e sua formatação via CSS) de um elemento HTML através do JavaScript.

Por se tratar de uma atividade de demonstração da mudança através do JavaScript as cores estão disponíveis apenas nesta página. Ao ir para outras páginas deste projeto voltam-se as cores normais.

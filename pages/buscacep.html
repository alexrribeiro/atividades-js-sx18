Busca de CEP

Utilizando biblioteca Ajax para se conectar a uma API de busca de CEP podemos, a partir do CEP encontrado preencher alguns campos do formulário abaixo. Quando você preenche o CEP e sai do campo (com TAB ou clicando em outro campo) a função para busca do CEP já é disparada e os dados deverão aparecer nos campos correspondentes.
